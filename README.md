# simpleSecondShallowModel
(I wanted the acronym to be triple S)

This started as a python implementation of Robin Hogan's model: https://www.met.reading.ac.uk/~swrhgnrj/shallow_water_model

but has since diverged a little bit. Mostly for teaching purposes.

The main thing is that I have tried to do things mostly functionally so that Numba works more or less automatically, with the goal of making it relatively fast.
